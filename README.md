# 🕐 Global Timezone Clock Application

A full-stack web application showing real-time time across 400+ world timezones.

## Features
- 400+ world timezones
- Real-time clock updates every second
- Search and filter functionality
- View 24 major world cities
- Responsive design for all devices

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Java, Spring Boot, MySQL
- **Database:** MySQL

## How to Run

### Frontend
```bash
cd frontend
python -m http.server 3000
# Open http://localhost:3000
